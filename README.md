## 目标-思路-代码
### 课程内容
```bash
开发工具 spring-tools-4使用
基于 Maven 的 MyBatis 逆向工程
父工程依赖管理
Spring 整合 MyBatis-配置数据源-配置AOP的事务
配置日志
配置spring-web-mvc,jsp和ajax
后台用户登录,md5加密
权限拦截器
基于注解的异常映射
分页实现
```
